# ☁️ Cloud Database Migration Simulator

An interactive front-end tool that simulates a real-world cloud database migration.  
It walks through schema export, data transfer, and validation phases — perfect for internship portfolio projects.

## Features
- Choose source (MySQL/PostgreSQL) and target cloud DB (AWS RDS, GCP Cloud SQL, Azure SQL)
- Full or incremental migration types
- Simulated progress bar, logs, and metrics
- Adjustable speed (slow to fast)
- No backend required — runs entirely in browser

## How to run
1. Download or clone this repository.
2. Open `index.html` in any modern web browser.
3. Select source/target and click **Start Migration Simulation**.
4. Watch logs and progress update automatically.

## Technology
- HTML5, CSS3, JavaScript (ES6)
- Simulated async delays
- Responsive design

## Use case
Demonstrate understanding of cloud migration strategies (ETL, CDC, validation) in a visual, interactive way. Great for cloud computing, DevOps, or database internship applications.
